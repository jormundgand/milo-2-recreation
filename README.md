# Milo 2 Recreated parts

This project contains my own recreated parts of the Milo 2. These are not meant to compete with the main milo project, but exists only because some of the parts were not easy to work with given the software I have avaialble.

The license will follow the Milo 2 parts license (GPL 3.0), since these are derived works, even if none of the original "source code" was used.
